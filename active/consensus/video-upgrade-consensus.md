## Top 10 — Rationale

### 1. Image-to-Video Dominance (8.14) — S-Tier
The single most impactful workflow shift. 90% of professional projects now start from an image, not a text prompt. This reframes the entire skill around image-first pipelines. Every framing scored it 7.8+. Non-negotiable inclusion.

### 2. 3 UGC Archetypes (8.08) — S-Tier
Walk-and-talk, driver's seat, at-home — these three templates cover the vast majority of UGC ad formats. Directly maps to ad revenue. Growth hacker gave it 8.7 (highest single score in the exercise). High integration ease because it's just prompt templates.

### 3. TikTok Slideshow Script Framework (7.81) — S-Tier
Already partially in the skill via campaign-runner. Completing this framework creates a direct pipeline from script to published content. Revenue scores were 9-10 across growth-oriented framings. Integrates cleanly with existing TikTok batch workflow.

### 4. Image Input First-Frame Control (7.76) — A-Tier
Companion to #1. Controlling the first frame via image input is the highest-leverage quality technique — it eliminates the "lottery" of text-only generation. First-principles thinker scored it highest (8.2) because it's a permanent craft upgrade.

### 5. Sora Prompt Structure (7.70) — A-Tier
Header, Shot Description, Technical Parameters, Audio — this 4-part structure is the grammar of video prompting. Extremely high integration ease (7-9 across framings) because it's a template pattern, not a tool dependency.

### 6. Seed Bracketing (7.45) — A-Tier
60% cost reduction by testing seeds before committing to full renders. Bootstrap operator scored this highest in the entire exercise (8.6). Pure operational efficiency — saves money on every single generation.

### 7. Two Image Workflows (7.37) — A-Tier
Objects need different treatment than characters in image-to-video. This distinction prevents the #1 mistake beginners make. Consistent 7.25-7.45 across all framings — no framing disagreed.

### 8. Weak-to-Strong Prompt Transformations (7.00) — A-Tier
Before/after prompt upgrade patterns. Teaches the skill to self-improve prompts rather than just execute them. First-principles thinker loved this (7.45) because it's a meta-skill that compounds over time.

### 9. Creative Director System Prompt Pattern (6.97) — A-Tier
A system prompt that turns the AI into a creative director rather than a prompt executor. High integration ease across all framings. Changes the skill's posture from "tool operator" to "creative partner."

### 10. Prompt Creativity Tradeoff (6.83) — A-Tier
Short prompts = more creative latitude. Long prompts = more control. Understanding this tradeoff prevents over-prompting (the most common mistake). Integration ease averaged 8.8 — it's just knowledge, no tooling needed.
