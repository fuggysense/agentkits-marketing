# Copywriter — Learnings & Patterns

> This file updates continuously. Add confirmed patterns, insights, and fixes discovered during use.

## Confirmed Patterns
- Ad creative quality is the main lever for lowering effective cost on Meta (CPM model)
- Hook must stop scroll in 3 seconds (video ads) or instantly (image ads)
- CTR < 1% = creative/headline problem — test new variations
- Frequency > 2-3 = creative fatigue — need fresh assets
- High CPM signals low engagement with current creative

## Diagnostic Rules
- Low 3-second views → Weak hook/opening — rewrite first 3 seconds
- Low ThruPlay (15s+) → Content not engaging enough after hook
- CTR below 1% → Test new headline and primary text
- Ad fatigue (frequency > 2-3) → Create new creative variations, same message

## Mistakes Not to Repeat
- Using AI-tell words: delve, comprehensive, robust, leverage, crucial, streamline, navigate, unlock, unleash
- Opening with generic phrases: "In today's fast-paced world..." / "Let's dive in!" / "Are you ready to..."
- Writing from feature level instead of 3-levels-deep emotional/financial payoff (So What? Chain)
- Same-length paragraphs, no rhythm variation — reads like a textbook
- Curiosity-only headlines without benefit = clickbait that disappoints
