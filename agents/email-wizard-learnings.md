# Email Wizard — Learnings & Patterns

> This file updates continuously. Add confirmed patterns, insights, and fixes discovered during use.

## Confirmed Patterns
- AI agent automation (email + WhatsApp) serves two purposes:
  - **1-to-1 selling:** Qualification and nurturing messages to move leads toward sales conversations
  - **1-to-many events:** Increase show-ups and commitment for webinars/presentations via automated reminders and pre-suasion
- These AI agent sequences directly impact show-up rates — 35% baseline, 40-45% with good sequences, 50%+ possible
- Show-up rate is the highest-leverage email metric in event funnels — small gains compound into major profit increases
- Pre-suasion sequences before events increase both show-up rate and closing rate
- AI agents replace manual texting/reminders for lead follow-up — saves 15+ hours/week

## Diagnostic Rules
- Show-up rate < 35% → Improve reminder sequences, pre-suasion content, AI agent automation (email/WhatsApp), webinar timing
- Show-up rate < 5% → Tech/setup issue (emails not sending, wrong links), not an optimization issue
- Low closing rate despite good show-ups → Add pre-suasion content before the event to warm attendees

## Mistakes Not to Repeat
- Weak reminder sequences that only send logistics (time/link) without building anticipation and pre-suasion
- Not automating follow-up sequences — manual texting doesn't scale

## Reference Data
- Webinar show-up rate benchmark: 35%+ (good), 40-45% (great with AI agent automation), 50%+ (exceptional at launch)
- Show-up rate improvement from 30% to 35% = +16% more attendees = compounds into 2.5x profit when combined with closing rate gains
