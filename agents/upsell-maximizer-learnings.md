# Upsell Maximizer — Learnings & Patterns

> This file updates continuously. Add confirmed patterns, insights, and fixes discovered during use.

## Confirmed Patterns
- Backend revenue is the real profit multiplier — front-end events are lead gen mechanisms
- Backend conversion rate benchmark: 5-10%
- Adding $18k-$20k backend to $3k front-end doubles average profit per client with zero extra ad spend
- Quarterly LTV growth strategy: add one new upsell/cross-sell product every quarter
- Example progression: $3k front-end → 1-on-1 coaching → mini mastermind → high-ticket program
- Higher LTV = more you can spend to acquire clients = outcompete everyone on ad spend

## Diagnostic Rules
- If LTV is only front-end price → Missing backend revenue, biggest untapped profit lever
- If competitor outspends you on ads → Their LTV is higher, you need backend offers to compete
- Profitable front-end but not scaling → Backend offer allows aggressive scaling by increasing what each client is worth

## Mistakes Not to Repeat
- Treating front-end revenue as the whole business — backend is where margins live
- Not introducing new offers quarterly — LTV should grow over time, not stay flat
- Trying to compete on ad spend alone without increasing LTV

## Reference Data
- Formula: `Avg Client Value = Front-End Price + (Backend Price × Backend Conversion Rate)`
- Example: $3k + ($20k × 10%) = $5k avg client value
- Doubling LTV from $3k to $6k = can spend 2x to acquire same client and still win
