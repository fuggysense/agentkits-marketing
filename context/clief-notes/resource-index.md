## The 60/30/10 Lens

| Category | Builds | Layer |
|----------|--------|-------|
| Official Skills | File creation, document processing | 30% Rules |
| Claude Code + Plugins | Code generation, project automation | All |
| MCP Servers | External tool integration, data access | Infrastructure |
| Remotion | Programmatic video rendering | 60% Code |
| Frontend/UI | Interface components, visualizations | 30% Rules |
| Learning/Reference | Understanding the tools | Foundation |
| Community Skills | Specialized workflows | 30% Rules |

Most repos give you 30% tools: repeatable patterns encoding rules. AI (10%) chooses which tool and fills in judgment. 60% layer is traditional code under the hood.

**Takeaway:** Knowing what tools exist > getting better at raw prompting. A mediocre prompt aimed at the right skill outperforms a brilliant prompt aimed at nothing.
