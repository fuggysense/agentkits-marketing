# Copywriting Learnings

Consolidated from [[copywriting]] and [[copy-editing]].

## Confirmed Patterns

- Headlines do 80% of the work — one headline can outpull another by 19.5x (Caples)
- Write like explaining to a smart skeptical friend — not a marketing team, not a guru
- Master formula: [Action verb] + [specific outcome] + [timeframe or contrast]
- The So What? Chain: ask "so what?" 3 levels deep on every feature to find the emotional/financial payoff
- Slippery slide (Sugarman): every element's job is to get them to read the next element
- Alternate rhythm: short punchy sentence -> longer breathing sentence -> short kicker. Repeat.
- Founder story arc: vulnerability -> credibility -> shared journey (highest-trust element on page)
- Testimonials need: [before state] + [action] + [specific outcome] + [timeframe] + [emotion]
- Disqualification ("you're NOT a fit if...") flips dynamic from "please buy" to "prove you're worthy"
- CTA formula: benefit-oriented text + [risk reversal] + [social proof] + [speed/ease] below it
- Match copy length to Schwartz awareness level: unaware = long copy, most aware = short

## Diagnostic Rules

- Copy sounds generic -> Run the So What? Chain 3 levels deep
- Reads like AI -> Check for: delve, comprehensive, robust, leverage, crucial, streamline, navigate
- No one reads past headline -> Test 5+ variations, lead with self-interest not cleverness
- Reader drops off mid-page -> Add seeds of curiosity, bucket brigades, vary paragraph length
- Sounds like a textbook -> Read aloud, add contractions, fragments, opinions, "I" and "you"
- CTA not converting -> Switch from command ("Sign Up") to benefit ("Start building")
- Testimonials feel weak -> Reformat as mini case studies with specific numbers and timeframes

## Mistakes Not to Repeat

- Using AI-tell words: delve, dive into, comprehensive, robust, cutting-edge, utilize, leverage, crucial, unlock, unleash, supercharge, game-changer, revolutionary, landscape, navigate, streamline
- Opening with: "In today's fast-paced world..." / "Let's dive in!" / "Are you ready to take X to the next level?"
- Same-length paragraphs (monotonous), every bullet starting the same way, hedging ("some may find")
- Curiosity-only headlines without benefit (clickbait that disappoints)
- Opening loops and never closing them — destroys trust
- Writing from feature level instead of emotional/financial bottom of So What? Chain
