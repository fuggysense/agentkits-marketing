# Meta Ads — Expert Operations Manual

> Load this reference whenever running, planning, or diagnosing Meta ad campaigns. This represents 20+ years of performance marketing expertise.

## Why Meta First (Platform Selection Logic)

- Meta = Facebook + Instagram simultaneously in one campaign
- Largest user base, best tracking, simplest interface
- Google Ads interface is complex by comparison
- TikTok/YouTube/LinkedIn come later — Meta teaches core principles
- Learn the numbers game here, apply everywhere else

## How Meta Charges (Pricing Model)

- Meta charges on **CPM (Cost Per 1,000 Impressions)** — NOT per click
- CPM driven by supply/demand of advertisers in your niche:
  - High competition (wealth, make money) = higher CPM
  - Low competition (parenting) = lower CPM
- CPM is largely uncontrollable — your lever is **ad quality** (better ads = more clicks per impression = lower effective cost)
- Never confuse with Google PPC — fundamentally different model

## The Full Funnel Metrics Chain

Every stage feeds the next — core mental model:

```
Ad Impressions
      ↓ (CPM)
Clicks to Landing Page
      ↓ (CPC — if too high, fix the AD)
Leads / Registrations
      ↓ (CPL — if too high, fix AD or LANDING PAGE)
Show-Ups
      ↓ (Cost Per Show-Up — fix REMINDERS + pre-suasion + AI agent automation (qualification + nurturing messages via email/WhatsApp))
Sales / Conversions
      ↓ (CPA — fix CLOSING RATE + PRESENTATION + OFFER)
Profit
```

**Key principle:** If any stage leaks, adding more ad spend amplifies the loss — does NOT fix the problem.

## Critical Benchmarks

| Metric | Benchmark |
|---|---|
| Landing Page Conversion Rate | 5-10%+ |
| CTR | Above 1% |
| Webinar Show-Up Rate | 35%+ (aim 40-45% with AI agent automation) |
| Closing Rate | 5-15% (varies by price point) |
| Backend Conversion Rate | 5-10% |
| ROAS | 3x minimum target |

- AI agent automation (qualification + nurturing messages via email/WhatsApp) can push show-up rates to 50%+ at launch
- Show-up rate below 5% = tech/setup issue, not optimization issue

## Pre-Launch Projection Framework

**Never skip this. Professionals build the math BEFORE spending a dollar.**

1. Set vision: e.g., 10 clients x $3,000 = $30k revenue
2. Work backwards through funnel:
   - Target sales -> needed show-ups -> needed leads -> needed ad spend
   - Example: 10 sign-ups -> 100 show-ups -> 250 leads -> ~$10k ad spend at $40 CPL
3. Map webinar plan: 4-6 webinars over 2 months
4. Calculate event cost per pax:
   - Free: office/education center partnership
   - ~$30/pax: learning centers ($1k ticket events)
   - ~$100/pax: hotel ($3k+ ticket events)
5. Calculate: Revenue - Ad Spend - Venue Cost = Profit
6. Ask: "Am I happy with this number?" If not, adjust BEFORE launching

**This is your stop-loss mechanism.** If projections show model is broken, stop before wasting five figures.

## Campaign Architecture

Never run one ad to one audience — like putting all money in one stock.

**Structure:**
- 5 audience sets x 2 ads minimum per campaign
- Test more audiences + more creatives simultaneously
- Monitor -> cut underperformers -> scale winners
- Naming: `[Date]_[Campaign Name]_[Audience/Interest Group]`

**Ad types:**
- ALWAYS run **Lead Ads** (not Traffic, not Boost Posts)
- Conversion objective = **Leads**
- Never run traffic ads — Meta optimizes for what you tell it

## Targeting Strategy

- Use interest-based targeting for directional signals
- Examples:
  - Business coaching -> business, marketing, lead gen, entrepreneurship, public speaking
  - Parenting -> parenthood, childcare, family
  - Health -> health, wellness
- Use AI to generate interest lists from: campaign topic, target audience, problem solved, offer
- Avoid hyper-narrow targeting — go broad, let Meta's algorithm optimize
- Broad targeting outperforms interest targeting once you have data

## Ad Launch Rules (Non-Negotiable)

| Rule | Why |
|------|-----|
| Always launch in the morning | Meta tries to spend daily budget by EOD; night launch = rushed, inefficient spend |
| Never set an end date | Keep ads running; pausing resets algorithm learning |
| Never pause a profitable campaign | Performance resets on restart |
| Always use Conversion/Lead campaigns | Never Traffic or Engagement |
| First campaign: allow 14 days | Setup mistakes + fixes; subsequent: 7 days per webinar |

## Budget Planning

- Meta only accepts **daily budget** (no total campaign budget)
- Formula: `Total Ad Spend / Days Running = Daily Budget`
- Example: $1,000 over 7 days = ~$143/day
- Minimum tests: $100-$200/day
- For 4-5 ad sets: ~$30-$50 per ad set per day

## Daily Tracking (4 Numbers)

Track daily: **Page Views, Leads, Show-Ups, Sales**

From Meta dashboard: CTR, CPL, CPC
From landing page: unique page views, conversion rate
Calculate manually: show-up rate (show-ups / leads), closing rate (sales / show-ups)

**Minimum 2 days data before optimizing.** Numbers stabilize after 48 hours.

### 1-to-Many (Webinar/Event) Campaign Decision Tree

```

   |

   | Yes

   | Yes

   | No

   |-- LP Opt-In Rate < 10% --> Landing Page Issue
   |     --> Ads CTR < 1%? --> Yes --> Continue running ads & tweak landing page
   |                        --> No --> Hit target CPL? --> Yes --> Continue
   |                                                   --> No --> (iterate)
   |-- Ads CTR < 1% --> Ads Issue --> Turn off campaign & relaunch new ads
```

### Full Optimization Decision Flowchart

After 2 days of sufficient spend:

```
Is campaign spending enough?
  --> No: Keep monitoring
  --> Yes:

  --> No: CTR below 1%? --> Fix the AD
        LP conversion below 5%? --> Fix LANDING PAGE
```

**Rule:** Fix the biggest leakage point first — don't spray optimization everywhere.

## The Compounding Effect

Small improvements compound across funnel:

| Metric | Before | After (+5%) | Impact |
|---|---|---|---|
| Show-Up Rate | 30% | 35% | +16% more show-ups |
| Closing Rate | 5% | 10% | 2x sales |
| Combined | — | — | **2.5x gross profit** |

10% improvement across all stages = **4-6x profit increase**.

## Three Campaign Scenarios

### A — Profitable but low closing rate
Fix webinar presentation first. Closing 5% -> 10% = CPA halves, profit doubles.

### B — Losing money
Diagnose biggest leak. Fix in order: LP (long vs short, headlines, CTAs) -> Show-up rate (reminders + AI agent automation) -> Closing rate (presentation + offer).
Example: Fixing CPL, show-up 30%->40%, closing 5%->8% = losing campaign -> $7,500 profit per event.

### C — Break-even
Improve multiple areas concurrently. Small gains across 3 areas = outsized compound results.

## Scaling Rules

**Only scale when profitable.** Scaling broken campaigns multiplies losses.

4 methods:
1. Increase daily budget (e.g., $60->$100/day)
2. Duplicate winning ad sets
3. Expand to new platforms (YouTube, TikTok) with same winning creatives
4. Expand to new countries — compare ROAS by country, allocate to winners

Every $1,000 not spent on a profitable campaign = opportunity cost of lost profit.

## Backend Revenue — The Profit Multiplier

Front-end event = lead gen mechanism. Real money is backend.

- Backend conversion: 5-10%
- Adding $18k-$20k backend to $3k front-end = **doubles average profit per client** with zero extra ad spend
- Formula: `Avg Client Value = Front-End Price + (Backend Price x Backend Conversion Rate)`
- Higher LTV = outspend competitors on ads while staying profitable

## Agency Red Flags

| Red Flag | What Good Looks Like |
|----------|---------------------|
| Reports vanity metrics (impressions, views) | Reports CPL, cost per show-up, CPA |
| Runs traffic ads | Runs lead/conversion ads |
| Says "spend more" when problem is conversion | Diagnoses which funnel stage leaks |
| Optimizes monthly | Optimizes after 48hrs of data |
| Can't quote expected CPL | Gives CPL estimate before launch |
| Never ran events themselves | Has direct event experience |

**Expose question:** "How much should I spend on ads, and what CPL should I expect?" No number = walk away.

## Professional Stop-Loss Protocol

- Set target metrics before launch
- When numbers fall outside thresholds: **kill campaign, don't spend more**
- Equivalent to trading stop-loss — prevents five-figure losses
- Never hope volume fixes a conversion problem. Fix leak first, THEN scale.

## Ad Metrics Quick Reference

**Standard Metrics:**
| Metric | What It Tells You | When to Fix |
|--------|-------------------|-------------|
| Amount Spent | Total money spent | — |
| Cost Per Lead | Cost for 1 contact | If high: Bad offer, weak LP, irrelevant targeting, weak copy/creatives |
| Cost Per Message | Cost for 1 conversation (Messenger/DM/WhatsApp) | If high: Bad offer, irrelevant targeting, weak CTA, weak copy/creatives |
| CTR | How engaging image & headline is | CTR < 1% = Test new headline |
| CPM | Cost to show ad to 1,000 people | High CPM = Low engagement image/ad |
| Frequency | Times same user sees ad | > 2-3 = ad fatigue = change image |

**Video Metrics:**
| Metric | What It Tells You | When to Fix | Formula |
|--------|-------------------|-------------|---------|
| 3-Second Views | Is hook stopping scroll? | Low = Weak hook/start | (3s Plays / Impressions) x Video Plays at 100 |
| ThruPlay | Watching 15+ seconds? | Low = Content not engaging | — |
| Cost per ThruPlay | Ad efficiency | High = Weak video/targeting | — |
| Average Watch Time | Engagement duration | <10s = Re-edit video | — |

## Scaling Beyond Ads (Team & Systems)

Hire in order of highest leverage:
1. Ad manager (when creating creatives + tracking consumes your time)
2. Admin assistant (when admin consumes client-facing time)
3. AI agents for lead follow-up (saves 15+ hours/week)

**Quarterly LTV Growth:**
Every quarter, add one upsell/cross-sell. Example: $3k front-end -> 1-on-1 coaching -> mini mastermind -> high-ticket program. Doubling LTV = doubles ad spend capacity = structural competitive moat.
