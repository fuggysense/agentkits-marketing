# Website Design — Learnings

## Confirmed Patterns
- Taste parameters default 6/5/5 for marketing pages; conversion-focused pages cap DESIGN_VARIANCE at 6
- Texture profiles layer ON TOP of cinematic presets — pick a preset for color/font/mood, then a texture for surface quality. They're not alternatives to presets.
- Anti-card-overuse (banned pattern L2) only applies when VISUAL_DENSITY ≤ 7; cards-in-grid is acceptable at higher density or for 6+ items
- Centered hero ban is conditional: banned at DESIGN_VARIANCE > 6, fine for conversion pages at lower variance
- Design Quality Score minimum 3.0 average required before declaring a build done
- Mode C redesigns should fix in priority order: Font swap → Color → Hover states → Layout → Components → States → Typography polish (highest visual impact per effort)
- Completeness protocol: never output `<!-- rest of sections -->` or skeleton HTML — count sections upfront, verify all present
